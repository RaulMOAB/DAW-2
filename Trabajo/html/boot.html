Firmware

  1. Firmware es un software que maneja físicamente al hardware.
  2. La BIOS de una computadora es un firmware cuyo propósito es activar una máquina desde su encendido y preparar el entorno para cargar un sistema operativo en la memoria RAM y disco duro.
  3. Firmware programa informático que establece la lógica de más bajo nivel que controla los circuitos electrónicos de un dispositivo de cualquier tipo.
  4. Es el software que tiene directa interacción con el hardware, siendo así el encargado de controlarlo para ejecutar correctamente las instrucciones externas.

BIOS

  1. BIOS: Basic Input / Output System, inventada por IBM.
  2. Pequeño programa que hace unos test básicos de Hardware al arrancar y le pasa el control al gestor de arranque.
  3. Es muy viejo, se dejó de fabricar por allá los 2010-2011.
  4. Tiene muchas limitaciones. Exige que el S.O esté en un disco con una tabla de particiones en formato MBR (Master Boot Record).
  5. No programable.

UEFI

  1. Universal EFI = EFI 2.0.
  2. EFI: Extensible(Programable) Firmware Interface.
  3. Sucesor de la BIOS. Es moderno.
  4. Exige que el S.O esté en un disco con tabla de particiones en formato GPT (GUID Partition Table). (GUID = Global Unique ID)
  5. Tiene modo de compatibilidad . en su mayoría, BIOS (Legacy).

En conclusión siempre arrancaremos en modo UEFI y los discos tendrán tablas de partición GPT. P ara que arranque siempre en modo UEFI hay que habilitarlo en la UEFI, también hay que desactivar el Secure Boot para poder arrancar ISO diferentes a Windows, Microsoft firma sus SO con un número hash y cada So tiene una firma distinta, por ello desactivando está opción podemos arrancar cualquier SO
