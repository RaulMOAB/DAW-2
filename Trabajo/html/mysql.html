MySQL

¿Que es MySQL?

MySQL es un sistema de gestión de bases de datos relacionales de código abierto respaldado por Oracle y basado en el lenguaje de consulta estructurado (SQL). MySQL funciona prácticamente en todas las plataformas, incluyendo Linux, UNIX y Windows. MySQL se asocia más a menudo con las aplicaciones web y la publicación en línea.



¿Como funciona MySQL?

MySQL se basa en un modelo cliente-servidor. MySQL funciona junto con varios programas de utilidad que soportan la administración de las bases de datos MySQL. Los comandos se envían a MySQLServer a través del cliente MySQL, que se instala en una computadora.
